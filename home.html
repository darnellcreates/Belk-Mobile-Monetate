    Experiences

    Experiences are campaigns that include specifics such as Who should be targeted, What content to display, When to display the content & Why the campaign is running. Monetate mobile support will include: Department/Landing Pages, Promo Detail Pages, Product Detail Pages and Assortment Pages.

    • Who? (Target Audience)
    • What? (Content To Show)
    • When? (Start Date To End Date)
    • Why? (Reason for Experience)

    Actions

    An Action is a directive to target a ID or Class within a HTML element. It can target img, divs, header elements and more

    Action Types


    Action Types are utilize to add engaging content onto the site, including adding HTML content, images, CSS, Javascript and more to an existing page.

    Target Types


    Target Types helps hone in on the desired audience. A deep dive into possibilities include location, browser, weather, customer behavior

    Tiered Mobile Coupons
